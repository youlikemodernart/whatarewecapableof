Boundary

  • This page is the live Fin home for the local finance dashboard feature model.
  • The hosted dashboard reads Fin invoices and approved derived finance summaries. It does not read raw `.finance/` files from Vercel.
  • Stripe Checkout links are created only from the Invoice Studio admin action. Client sends, refunds, customer changes, Mercury/Gmail/bank/card writes, and transaction-category writes require separate approval gates.
