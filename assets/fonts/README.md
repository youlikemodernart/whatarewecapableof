# Geist Mono fonts

Self-hosted WOFF2 files for the `Geist Mono` CSS family used by the WAWCO legacy token stack.

Source: Next.js package dev-overlay font files, copied locally so production pages do not depend on external font origins.

Files:

- `geist-mono-latin.woff2`
- `geist-mono-latin-ext.woff2`
